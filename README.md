**Scanner**

1. 정규 표현을 이용해 구문 분석과 데이터 파싱하는 작업 수행

2. 동기화 되지 않아서, 멀티 스레드에 안전하지 않다.

3. 버퍼 크기가 작다.(1KB)

4. 개발자가 Exception Handling 하지 않는다. 


**BufferedReader**

1. String만 읽을수 있다. 
2. 동기화 되어 있어, 멀티 스레드에 안전 
3. 버퍼크기가 크다. (8KB)
4. 개발자가 Exception Handling 해야한다.
